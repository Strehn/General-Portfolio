
   ----- IMPORTANT INFO -----
*  Grammars can only have letters and every production must be only one letter.

*  Grammars must be in Chomsky Normal Form

*  Grammar put into Grammar.txt and then changed within the file

*  User enters the grammar's starting variable, and number of productions
    * This makes coding this easier because computer knows what to look for
    
*  Grammars MUST Look like this (NO SPACES)
    S->AB|BC
    A->a
    B->b
    C->AB
